---

## 📌 프로젝트 소개

본 GitHub Repository는 **모두의 연구소 데이터 사이언티스트 5기 교육과정**에 참여하며 작성 및 제출한 모든 학습 자료와 결과물을 보관 및 공유하기 위해 개설하였습니다. 

이곳에 업로드된 소스 코드와 문서들은 교육 기간 동안 요청받은 **공식 제출물**들을 포함하고 있습니다.

## 📚 학습 자료 구성 및 커리큘럼 로드맵

이 저장소는 주로 최종 제출물(코드)을 보관하며, 상세 학습 노트는 별도의 Colab 환경에서 관리됩니다.

| No. | 학습 노드 타이틀 | 주요 내용 |

| 3 | **비정제 데이터 체험** | 비정형/준정형 데이터 처리 및 탐색 |
| 4 | **기초통계** | 데이터 분석을 위한 통계적 기본 개념 |
